🚀 Autostart (systemd)

Jalankan perintah ini di terminal HG680P agar Bell System otomatis aktif saat boot:

1. Install service 

sudo cp /opt/bell/bell.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable bell
sudo systemctl start bell

2. Cek status 

sudo systemctl status bell
sudo journalctl -u bell -f

3. Install dependensi sistem 

sudo apt install mpv mpg123 alsa-utils -y
pip3 install -r /opt/bell/requirements.txt
